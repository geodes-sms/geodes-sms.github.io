  • Open collaborative data engineering
    • Prof. Dirk Riehle – Friedrich-Alexander-Universität Erlangen-Nürnberg
    The JValue project investigates open collaborative data engineering: We are trying to take the world of open-source software engineering to data engineering.
    The main components of the JValue project are:
    1. Jayvee: The domain-specific language for modeling data pipelines
    2. JV Hub: A software forge for collaboration in data engineering
    3. JV Cloud: A cloud execution infrastructure for data pipelines modeled using Jayvee

    In this talk, I will focus on Jayvee, the language, and its integration with JV Hub and JV Cloud.
    Vendredi 1 décembre 2023 @11h30 3195, Pav. André-Aisenstadt
